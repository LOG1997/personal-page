# Vue3 Personal Page个人主页 🦓

[项目网址](https://24years.top)
> 使用vue3+ts+sass搭建

## 运行项目
克隆项目到本地
```
git clone https://github.com/LOG1997/vue3-personal-page.git
```
安装依赖
```
cd vue3-personal-page
npm i
```
运行项目
```
npm run dev
```
